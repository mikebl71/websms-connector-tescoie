Support

If you have any questions about this application or experience some problems, first of all, have a look at the FAQ section. It might already be addressed there.

Issues

  1. If you have problems with sending messages, first, login to www.tescomobile.ie website and check that you can send messages from there. It does happen once in a while that this site goes down for various reasons or Tesco has problems with sending webtexts. If you experience problems there, you can contact Tesco Mobile's customer support.
  2. If all works fine there, then there indeed might be a problem with the application. You can then email me and explain the issue.

NOTE that leaving negative comments in the app store does not really help with resolving any issues. So, if you experience problems, please try to follow the above steps first and then we can actually fix them.

Questions and comments

For general questions and comments, you can email me.
