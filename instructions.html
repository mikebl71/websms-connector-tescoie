Instructions

Installation

This android app is an extension to WebSMS. So

  • First, if you don't already have it, install WebSMS from Google Play or from whatever app store you prefer.
  • Then, install the WebSMS TescoIE Connector from Google Play or from whatever app store you prefer.

When you start WebSMS the next time, you'll be able to configure the TescoIE Connector as described below. Note that you must configure it with your account details before you can use it.

Configuration

To configure WebSMS TescoIE Connector, you first need to get to its preferences screen. For this,

  1. Start WebSMS
  2. Press Menu
  3. Select Preferences » Preferences - tesco

All connectors are installed in the disabled mode. You must enable the Connector to use it.
To do this, tick Enable tesco.

Now you need to configure your account details for www.tescomobile.ie. For this, provide the following values:

  • Mobile Number  - Your Tesco Mobile phone number (as you enter it when you login to www.tescomobile.ie, e.g. 0891234567)
  • Password  - Your password for www.tescomobile.ie website
